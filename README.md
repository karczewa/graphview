# GraphView

A web application that connects to a Neo4j graph database and renders its data as an interactive graph visualization with configurable colors and geometric shapes per node label and relationship type.

## Tech stack

- **Frontend**: React + TypeScript, D3.js, Vite, Tailwind CSS, Zustand
- **Backend**: Node.js + Express + TypeScript
- **Database**: Neo4j (via HTTP API)
- **Testing**: Vitest + React Testing Library
